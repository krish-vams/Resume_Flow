# Demo Video Flow

Use this sequence for a 3-5 minute portfolio demo.

## Script

1. Start at `/login` and sign in.
2. Open `/settings/profile` and show candidate profile data.
3. Open `/jobs/new`, paste a job description, and save it.
4. Open the job detail page and run JD analysis.
5. Show eligibility status, extracted keywords, and recommended focus.
6. Open `/prompts`, select a prompt, and assemble/copy the final prompt.
7. Return to the job detail page and upload a raw Gemini DOCX resume.
8. Validate the resume and show the validation checklist.
9. Format the resume and download the final DOCX.
10. Open `/resumes` and show the resume library entry.
11. Open `/applications` and mark the job as applied.
12. Return to `/dashboard` and show updated metrics.
